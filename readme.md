<h1 align="center"> FedAPI </h1>
 <p align="center"> API for Fed </p>

FEDapi is a modular backend API platform designed to power digital game distribution services.
The platform focuses on security, scalability, and operational readiness, providing essential services such as authentication, catalog management, cart and order processing, payment orchestration, and entitlement-based game library management.
