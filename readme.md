<h1 align="center"> FedAPI </h1>

<h3 align="center">
FedAPI is a backend API platform designed to power a digital game store ecosystem.
</h3>

---
The API will provide services that covers overall function of Fed, such as

- Authentication
- Catalog
- Game Store
- Cart System
- Ordering process
- payment authenctication and system
- user game library
- etc


---

<h3>Tech Stack</h3>

---
this system using NodeJS, ExpressJS, and PostgreSQL


<h3> MVP Scope </h3>
The first version of FedAPI focuses on the minimum backend services required to operate a digital game store.
- Core modules planned for MVP:
- Authentication
- User Profile
- Game Catalog
- Shopping Cart
- Order
- Payment Authentication

---

Clone Repository
git clone https://github.com/RolloPanda/FedAPI
cd FedAPI

Install Depedencies
npm install

Run Development Server
npm run dev
